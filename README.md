# NHS RTT Waiting Time Analysis
## Project Overview
This project analyses NHS Referral to Treatment (RTT) waiting-time data to investigate waiting-time distributions, assess reporting completeness, and explore variation across NHS providers and treatment functions. The project uses publicly available NHS RTT data [found here](https://www.england.nhs.uk/statistics/statistical-work-areas/rtt-waiting-times/rtt-data-2025-26/). 

The analysis was developed as a portfolio data science project demonstrating:
- data cleaning and validation
- exploratory data analysis
- feature engineering
- handling messy administrative datasets
- statistical reasoning
- data visualisation
- reproducible analytical workflows in Python

## Objectives
The main objectives of the analysis were to:
- Understand the structure of NHS RTT waiting-time reporting data
- Validate relationships between reported totals and waiting-time buckets
- Assess reporting completeness and missingness patterns
- Engineer clinically meaningful waiting-time metrics
- Explore provider-level variation in long waiting times


## Dataset
The dataset contains NHS RTT waiting-time pathway data split across weekly waiting-time buckets.
Key variables include:
- Provider organisations
- Commissioner organisations
- Treatment functions/specialties
- Weekly waiting-time distributions
